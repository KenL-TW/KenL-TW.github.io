# KB 起始導覽

這份知識庫提供網站導覽、專案摘要、履歷重點與常見問答，供 Agent 與讀者快速查詢。

## 建議閱讀順序
1. 個人定位：`kb/docs/01_about-ken.md`
2. 技能摘要：`kb/docs/02_skills.md`
3. 專案入口：`kb/docs/04_projects.md`
4. 履歷入口：`kb/docs/05_resume.md`
5. 架構入口：`kb/docs/06_architecture.md`
6. Agent 規範入口：`kb/docs/07_agent-governance.md`
7. 常見問題：`kb/docs/10_faq.md`

## 查詢原則
- 以可驗證內容為優先，避免推測。
- 涉及數據、日期、技術版本時，優先回覆知識庫已記載資訊。
- 如知識庫未覆蓋，需明確標示「目前無資料」。

## 維護節奏
- 新增作品或文章後，需同步更新：`kb/projects/*`、`kb/index.json`、`sitemap.xml`。
- 建議每月進行一次知識庫盤點與文字一致性校正。
- 維運細節請參考：`kb/docs/13_kb-maintenance.md`。

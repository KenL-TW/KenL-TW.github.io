# Resume Entry（履歷入口）

本頁為履歷區導覽入口，提供摘要、完整版本與能力對照。

## 主要文件
- 履歷摘要：`kb/resume/resume_summary.md`
- 完整履歷：`kb/resume/resume_full.md`
- 成果亮點：`kb/resume/achievements.md`
- 技能矩陣：`kb/resume/skills_matrix.md`

## 使用情境
- 快速認識背景：先看 `resume_summary`
- 面試與查核：搭配 `resume_full` + `achievements`
- 職能對照：查看 `skills_matrix`

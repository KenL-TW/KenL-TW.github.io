# Glossary（詞彙表）

## SA（System Analyst）
系統分析師，負責把商業需求拆解成系統規格、流程與資料模型。

## SOW（Statement of Work）
工作說明書，定義交付範圍、責任邊界、時程與驗收條件。

## Serverless
以雲端託管執行環境為主的架構模式，常見組合為 API Gateway + Lambda。

## KB-first
先從知識庫檢索可驗證內容，再由模型組織語句的回覆策略。

## Tool-first
先執行工具與檢索，再進入語言生成，降低幻覺與錯誤引用。

## Deterministic Retrieval
使用固定規則進行文件切塊與排序，確保檢索結果可重現。

## Guard
在最終回覆前做一致性檢查，移除未被知識來源支持的敘述。

## PDF Sign Editor
瀏覽器端的 PDF 標註與簽章工具，支援印章、手寫簽名、文字與框線。
